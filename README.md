MyStuffLabelled – Frontend 
Frontend for MyStuffLabelled, a personal inventory application that allows users to manage items and generate QR codes linking to stored item information.

##Features
-User authentication (login / signup)
-Create, edit and delete items
-Display purchase metadata and receipt images
-QR code generation and printable labels
-Item detail page accessible via QR scan
-User-specific data separation
-Simple and responsive UI

##Tech Stack
-Vue.js
-JavaScript
-HTML / CSS
-Axios
-Vue Router
